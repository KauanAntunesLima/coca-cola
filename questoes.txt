repertorio Git é um local onde fica armazenado projetos, e ele é muito importante pois nos permite fazer rastreamento de alteraçÕes e reverter mudanças se necessário

Qual a função do Git Bash?

Git Bash é um terminal de linha de comando que fornece uma interface para usar o Git no sistema Windows. Ele simula um terminal Unix, permitindo que você execute comandos Git

O que faz o comando git add e qual a diferença entre git add <arquivo> e git add .?

O comando git add é usado para adicionar alterações ao índice do git.
git add <arquivo> adiciona um arquivo específico para a área de staging.
git add . adiciona todos os arquivos modificados no diretório atual e subdiretórios para a área de staging.

Por que utilizamos o ponto (.) no comando git add .?

O ponto (.) representa o diretório atual, e ao usá-lo em git add ., você está pedindo ao Git para adicionar todos os arquivos modificados dentro do diretório atual e seus subdiretórios para a área de staging.

Explique o comando git commit e por que é importante criar uma mensagem descritiva.

O comando git commit é usado para salvar as alterações que você fez no código, ele cria um ponto de verificação que você pode voltar no futuro, caso precise.
Quando fazemos um commit, é importante adicionar uma mensagem descritiva que explique o que foi mudado, isso facilita entender o que foi feito em cada commit

Qual a função do comando git push?

O comando git push é usado para enviar as alterações que você fez no seu repositório local para um repositório remoto como o GitHub que usamos em aula.

Qual a importância de criar commits frequentes durante o desenvolvimento de um projeto?

Fazer commits frequentes é importante porque ajuda a manter o controle sobre o que foi feito no código, tornando mais fácil identificar problemas e desfazer mudanças.

Explique a função das tags HTML e como elas estruturam uma página.

As tags HTML são usadas para marcar e estruturar o conteúdo de uma página web. Elas informam ao navegador como exibir o conteúdo,
 como texto, imagens, links, tabelas, entre outros. Cada tag tem uma função específica, por exemplo, a tag <h1> é usada para títulos, <p> para parágrafos e <img> para imagens. 

O que são atributos em uma tag HTML e qual a sua função?

Atributos em uma tag HTML são informações adicionais que modificam o comportamento ou a aparência do conteúdo dentro da tag.
 Eles são colocados dentro da tag de abertura, geralmente no formato nome="valor". Por exemplo, a tag <img src="imagem.jpg" alt="Descrição da imagem">

O que é o Emmet e como ele facilita o desenvolvimento de páginas HTML?

Emmet é uma ferramenta que facilita a escrita de código HTML e CSS, permitindo criar estruturas de forma mais rápida. 
Com ele, você pode digitar abreviações e o Emmet as expande automaticamente para tags HTML completas.
